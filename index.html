Projects

Web Development

Nerd'Store is a specialized online bookstore catering to fans of science fiction, fantasy, graphic novels, manga, young adult fiction, and more. The platform aims to provide a seamless and enjoyable shopping experience for book lovers, with various features to explore and utilize. As a Full-Stack Developer in the Nerd'Store project, I was responsible for developing the user interface, implementing the secure checkout process, and designing the database schema. I collaborated with the team to ensure a cohesive and user-friendly design, integrated third-party projects and APIs to enhance website functionality, and conducted user testing to incorporate feedback for an improved user experience. For the frontend, we utilized HTML, CSS, JavaScript, and React to create a dynamic and responsive user interface. The backend was built using Node.js and Express, providing a robust server environment to handle user requests and data processing. We used MongoDB for the database, allowing for efficient storage and retrieval of a large and diverse book catalog. Authentication was managed using JSON Web Tokens (JWT) to ensure secure handling of user data. Stripe was integrated for secure payment processing, ensuring customer confidence during transactions.
